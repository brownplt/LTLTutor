Globally Review

It may be the case that you believe the G (Globally) operator implicitly applies at some state, meaning it automatically applies at a particular point in time. However, the Globally operator has a stronger meaning: it asserts that the proposition x must hold true at all future states, starting from the current state onwards. On the other hand, simply stating the proposition x only requires that x be true in the current state, placing no constraints on future states.

For example, if x represents 'the system is secure', the statement x means that the system is secure in the current state, but doesn't guarantee that it will remain secure in the future. G x, on the other hand, states that the system is secure, and will remain secure in every possible future state.
