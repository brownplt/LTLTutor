Globally Review

In LTL, the formula G x means that the proposition x is always true from the current state onwards, not just in the current state. On the other hand, simply stating the proposition x only requires that x be true in the current state.

For example, if x represents 'the system is secure', then G x would mean that the system must be secure at all times, in every possible future state. On the other hand, x would mean that the system is secure in the current state, but it may not be secure in the future.
