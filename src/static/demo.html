Key Features Implemented:
  • Matrix Layout: Time steps as columns, subformulae as rows
  • 0/1 Values: Clear binary satisfaction values
  • Color Coding: Green for satisfied (1), red for unsatisfied (0)
  • Pattern Recognition: Easy to spot relationships like "Xa" being "a" shifted left
  • Complete Coverage: All subformulae shown, from atomic to complex
